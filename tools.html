Tools of the Trade

I use a number of tools to make my work, large and small. Most are inexpensive and easy to obtain. Here are a few of my go-to tools.

Sander: This is a belt sander, which is probably the most expensive tool in the bucket. It can be used to put a smooth finish on wide surfaces.

Triangular file: This file can be used to get a smoot finish and a sharp shape on tiny little corners and crevices.

Chip-carving pocket knife: This one has a large and small blade.
While it's made for chip-carving, I use it for all my whittling.

Coping saw: This saw makes it easy to cut complex shapes. The blade bends and rotates so that you can follow an outline while cutting.
